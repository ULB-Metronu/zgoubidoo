ZgoubiMpl¶

class zgoubidoo.vis.ZgoubiMpl(ax=None, with_boxes: bool = True, with_frames: bool = True, with_centers: bool = False, tracks_color: str = 'b', **kwargs)[source]¶

Bases: zgoubidoo.vis.zgoubiplot.ZgoubiPlot

A matplotlib implementation of a ZgoubiPlot artist.

Parameters:ax – the matplotlib ax used for plotting. If None it will be created with init_plot

(kwargs are forwarded). :param with_boxes: draw the body of each elements :param with_frames: draw the entry and exit frames of each elements :param with_centers: draw the center of each polar coordinate elements :param tracks_color: color for the plotting of tracks :param kwargs: forwarded to ZgoubiPlot and to init_plot.
