zgoubidoo.twiss.compute_transfer_matrix(beamline: zgoubidoo.input.Input, tracks: pandas.core.frame.DataFrame, align_on: str = 'X') → pandas.core.frame.DataFrame[source]¶

Constructs the step-by-step transfer matrix from tracking data (finite differences). The approximation uses the O(3) formula (not just the O(1) formula) and therefore makes use of all the particles.

Parameters:
  • beamline – the Zgoubidoo Input beamline
  • tracks – tracking data
  • align_on – coordinates on which the tracks are aligned (typically ‘X’ or ‘S’)
Returns:

a Panda DataFrame representing the transfer matrix

Example

Here is a typical example to call compute_transfer_matrix:

>>> tracks = zgoubidoo.read_plt_file()
>>> zi = zgoubidoo.Input()
>>> matrix = zgoubidoo.twiss.compute_transfer_matrix(zi, tracks, align_on='X')
